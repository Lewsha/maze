Программа для поиска кратчайшего пути в лабиринте. 
Абстрактно лабиринт сделан в виде графа, где каждая клетка - вершина. 
Файл описывает лабиринт следующим образом. Сначала в строчках для каждой клетки указываются клетки, с которыми у нее есть связь. Затем после ключевого слова end в трех строчках указываются количество бомб, номер клетки входа и номер клетки выхода.
Запуск осуществляется через консоль:
<интерпретатор python> main.py <файл лабиринта>
